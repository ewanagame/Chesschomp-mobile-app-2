## Stockfish chess engine

This app uses **[Stockfish 18](https://github.com/official-stockfish/Stockfish)** (GPL-3.0) as an on-device analysis engine.

- **Official source:** [github.com/official-stockfish/Stockfish](https://github.com/official-stockfish/Stockfish) (submodule at `vendor/Stockfish`, tag `sf_18`)
- **Runtime build:** WebAssembly port from [nmrugg/stockfish.js](https://github.com/nmrugg/stockfish.js) release v18.0.0 (`stockfish-18-lite-single`), bundled offline in `assets/stockfish/`
- **Integration:** The engine runs in an isolated hidden WebView and communicates with the app via UCI text over `postMessage` only

Stockfish is Copyright the Stockfish developers (see `vendor/Stockfish/Copying.txt`). The engine is not linked into the app's native binary.

## Opening book (ECO)

Move classification uses a bundled opening database derived from the public **[lichess-org/chess-openings](https://github.com/lichess-org/chess-openings)** dataset (CC0 / public domain).

- **Bundled file:** `assets/openings/eco-book.json` (generated by `npm run openings:sync`)
- **Lookup:** Prefix trie keyed by SAN move sequence for offline book detection
